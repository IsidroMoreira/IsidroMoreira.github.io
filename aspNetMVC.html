ASP.NET (MVC6)


This section its dedicated to Asp.Net, my internship was 60 days working in asp.net,
it was a short time, but I managed to leave something functional.
I will leave some videos and comments about what it was supposed to do and how it works.


The work The assignment was to create a website to manage customers, distributors, products, contacts and licenses.
The database was created with code first approach, entity Framework and scaffold.
For the design i used bootstrap and data tables.
We see in the videos that some fields are created dynamically with the press of a button, used a JavaScript function to change the indexes whenever a field is deleted, to be able to save it to the database.
I used SQL with relational tables, the connection to the database used a repository pattern.
